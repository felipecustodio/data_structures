# Counting Inversion

### Input Format
* First line consist no. of test cases T.
* Next line contains N, no. of elements in array.
* Next line contains N integers, elements in array.

### Output Format
> Number of inversions in array.

### Sample Testcases:
Input:  
```
1
5
2 3 8 5 1
```

Output:
```
5
```
